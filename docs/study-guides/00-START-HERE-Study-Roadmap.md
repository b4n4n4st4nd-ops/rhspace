# Ryan Hambleton — Web Development Study Roadmap

**Purpose:** Learn enough to customize ryanhambleton.space, build React sites professionally, and manage AI agents that generate web code.

**Your stack:** Domain (Vercel) → GitHub → Next.js (React) → JSON/MDX content → live at ryanhambleton.space

---

## How to use these guides

1. Read guides **in order** (00 through 09).
2. After each guide, open your project folder and **find the files mentioned**.
3. Make **one small edit** per study session and run `npm run dev` to see the result.
4. Add notes to `docs/learning-journal.md` in your repo.

Upload the `.docx` files from `docs/study-guides/word/` to Google Drive for offline reading.

## The three skills employers care about

1. **Can you ship?** — Site live on a domain, updates via Git (Guide 06).
2. **Can you maintain?** — Read React/Next code, fix bugs, add pages (Guides 03–05).
3. **Can you integrate?** — APIs, data, auth basics (Guide 07).

---

## Your project at a glance

```
ryanhambleton-space/
  app/           → Pages (URLs)
  components/    → Reusable UI pieces
  content/       → Your words & data (JSON, MDX)
  public/        → Images, resume PDF
  lib/           → Code that loads content
  docs/          → Documentation (including these guides)
```

**Commands you need:**

- `npm run dev` — run site locally (localhost:3000)
- `npm run build` — test production build
- `git add .` → `git commit -m "message"` → `git push` — publish updates
