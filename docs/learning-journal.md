# Learning Journal

"DNS translates names to IPs. Domain is the address; hosting is the server. HTTPS encrypts traffic. localhost = dev on my machine; .space = live on Vercel. Sites are HTML + CSS + JS."

**Guide 02 — checkpoint (journal)**

1. **JSON vs JavaScript:** JSON is data text; JavaScript is code that can read JSON and run logic.
2. **`{site.email}`:** Curly braces in JSX insert a value from your data — here, your email from `site.json`.
3. **TypeScript:** JavaScript plus types so mistakes get caught before the site runs.
4. **npm dependencies:** Listed in `package.json` at the project root.
5. **`NEXT_PUBLIC_`:** That prefix sends values to the browser, so API keys must never use it.
