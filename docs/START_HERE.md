# START HERE — Your learning path

You own **ryanhambleton.space** for 12 months. This folder is the **code** for your site. This guide is the map.

## The 4 layers (bottom to top)

1. **Domain** — the address (`ryanhambleton.space`) — you have this on Vercel
2. **Code** — this folder — pages, text, colors, images
3. **Hosting** — Vercel keeps the built site online — **not connected yet**
4. **Browser** — what visitors see when they type your domain

## Milestone ladder

| # | Milestone | How to know you're done |
|---|-----------|-------------------------|
| 1 | Local app works | `npm run dev` → open http://localhost:3000 |
| 2 | You edited a file | Change text in `content/site.json`, see it in browser |
| 3 | Git commit | `git status` shows clean after commit |
| 4 | GitHub backup | Code visible on github.com |
| 5 | Vercel deploy | You get a `*.vercel.app` URL |
| 6 | Custom domain | `ryanhambleton.space` shows your site |
