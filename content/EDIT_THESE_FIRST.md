# Edit these files first (Module 1)

Before deploying, personalize these files. No React knowledge required.

## `content/site.json`

- `tagline` ‚Äî your one-line pitch (shows on homepage)
- `email` ‚Äî contact email (footer, resume)
- `social.linkedin`, `social.github`, `social.tableau` ‚Äî your real profile URLs
- `nav` ‚Äî add/remove menu items (creates links only; you still need a matching `app/.../page.tsx` for new pages)

## `content/resume.json`

- `education` ‚Äî replace "Your University" placeholder
- `experience` ‚Äî update companies, roles, highlights (use clean-room wording for employer work)
- `skills` ‚Äî tools you want to highlight

## `content/about.mdx`

- Your career story in markdown

## `content/portfolio/*.json` + `*.mdx`

- One pair per project: JSON = card metadata, MDX = STAR case study

See [docs/module-1-edit-content.md](../docs/module-1-edit-content.md) for the full exercise.
