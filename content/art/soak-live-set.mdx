## Concept

Stinky Blinkers / Possitopia was a participatory installation at SOAK — a Burning Man regional — blending LED sculpture, wizard mythology, tactile controls, music, and live audio programming. Light, sound, character, and interaction became one shared ritual-play environment.

## What participants experienced

- A dome installation with a wizard face and illuminated eyes
- Physical controls mapped to both light and sound behavior
- A live set where the sculpture felt playable — not just decorative

## Technical flow

```
LED Sculpture Controls
  ↓
Python Serial Listener
  ↓
OSC Messages
  ↓
Max for Live
  ↓
Ableton Instrument + Audio Effects
  ↓
Interactive Speaker / Live Set
```

## What I built

- **Environment** — Dome installation, wizard face, participatory art space alongside enso paintings
- **Interaction** — Physical controls mapped to sound and light behavior
- **Software** — Python serial listener and OSC message routing into Max for Live
- **Audio** — Custom Ableton instrument preset, mapped effects, and performance clips
- **Experience** — Live sound-reactive sculpture and participatory ritual

## Key insight

The installation translated physical light controls into musical changes — triggering instruments, shaping effects, and turning the sculpture into a playable sound environment.
